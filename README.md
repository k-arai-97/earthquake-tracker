# 地震検索システム (Earthquake Search System)

P2P地震情報APIを利用して、特定の期間内に発生した「震度3以上」または「マグニチュード5以上」の地震を抽出するデスクトップアプリです。

## 特徴
- **期間指定**: カレンダー形式で開始日と終了日を選択可能。
- **高精度なフィルタリング**: `isinstance` を用いた型判定により、APIのデータ不備（マグニチュード不明など）によるクラッシュを防いでいます。
- **最新API対応**: 日付形式（ハイフン/スラッシュ）の混在やミリ秒の有無を自動で吸収します。

## 使い方
1. Pythonがインストールされた環境で実行します。
2. 必要なライブラリをインストールします: `pip install requests`
3. `python あなたのファイル名.py` で起動。

## 開発の背景
APIの仕様変更（日付形式の変更やデータ型のゆらぎ）に対応し、確実に過去のデータを遡れるようにロジックを最適化しました。
