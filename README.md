# 地震データ収集アプリ (Earthquake Tracker)

P2P地震情報 APIを活用し、直近の大きな地震を抽出して表示するPythonデスクトップアプリです。

## 概要
このアプリは、APIのデータ保持期間（直近約14日間）を最大限に活用し、その中から**マグニチュード5.0以上**かつ**最大震度3以上**の重要な地震情報を自動でフィルタリングして表示します。

## 主な機能
- **自動期間設定**: 起動時に「今日から14日前まで」の検索期間を自動で算出・表示します。
- **特定条件の抽出**: 膨大な地震データの中から、被害の可能性がある規模（M5.0以上/震度3以上）のみを抽出します。
- **リアルタイム検索**: 100件ずつ遡ってデータをスキャンし、条件に一致するものを即座に表示します。

## 開発の背景（仕様について）
利用しているAPI（P2P地震情報）の仕様に基づき、現在は過去14日間のデータを対象としています。
当初は「取得できない過去の年数」を選択させる設計にしていましたが、**「確実に取得できる最新データ」**を正確に分析することに特化した設計に変更しました。

## 実行環境
- Python 3.x
- 必要なライブラリ: `requests`, `tkinter`

## 使い方
1. アプリを起動します。
2. 画面に表示された「取得対象期間」を確認します。

3. 「検索開始」ボタンを押すと、分析が始まります。
